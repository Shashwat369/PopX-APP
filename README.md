# 🚀 PopX UI Assignment

A pixel-perfect implementation of the **PopX Mobile UI** using **React.js** and **CSS**, developed from the provided Adobe XD design.

---

🔗 https://pop-x-app-v6qf.vercel.app/

---

## 📂 GitHub Repository

🔗 https://github.com/Shashwat369/PopX-APP

---

## 📸 Preview

The application includes the following screens:

- Landing Page
- Login Page
- Signup Page
- Profile Settings Page

---

## ✨ Features

- Pixel-perfect UI based on Adobe XD
- React Router DOM for client-side routing
- Signup form validation
- Login form validation
- Dynamic navigation between pages
- Pass user data using React Router state
- Mobile-first design (375 × 812)
- Clean folder structure
- Reusable MobileContainer component

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- CSS3
- Vite

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Shashwat369/PopX-APP.git
```

Move to the project folder

```bash
cd popx-ui
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

## 📱 Application Flow

### Landing Page

- Navigate to Signup

- Fill user details

### Login

- Basic validation
- Navigate to Profile Settings

### Profile Settings

- Displays user information
